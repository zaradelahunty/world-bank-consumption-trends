# World Bank Consumption Trends Analysis

## Project Overview
This project analyses global consumption trends using World Bank API data. It compares global consumption with UK-specific trends over time to identify patterns and differences in economic behaviour.

## Data Source
- World Bank Open Data API
- Indicator: Household Final Consumption Expenditure

## Tools & Technologies
- Python
- pandas
- matplotlib
- REST API

## Key Features
- Extracted real-time data from the World Bank API
- Cleaned and structured raw JSON data
- Performed time-series analysis
- Compared UK consumption against global trends
- Visualised results using log-scale transformation

## Key Insight
Global consumption has grown significantly over time, while UK consumption shows steady but proportionally smaller growth compared to global trends.

## Example Output
The project generates a visual comparison of global vs UK consumption trends over time.
